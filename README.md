# mg_ms_tflinter
TF Linter created for Mantel Group Managed Services

## Installation

### Install GO
```bash
brew install go
```

### Initialize GO
```bash
go mod init github.com/JMShadbury/mg_ms_tflinter
```

### Install tflint plugin
```bash
go get github.com/terraform-linters/tflint-plugin-sdk
```

### Install tflint
```bash
brew install tflint
```
